## All services

### Start all

`docker-compose -p jdufner -f docker-compose.yml up`

### Stop all

`docker-compose -p jdufner -f docker-compose.yml down`

## Jenkins

### Start Jenkins

`docker-compose -f docker-compose-jenkins.yml up`

### Stop Jenkins

`docker-compose -f docker-compose-jenkins.yml down`
